# Documentation

This folder contains learning, planning, and workflow documents for the local Bitcoin Core `regtest` setup.

## Documents

- `learning-goals.md` defines what to learn from this project and local RPC practice.
- `app-rpc-contract.md` defines the planned contract between the app layer and Bitcoin Core RPC.
- `rpc-learning-path.md` defines a practical command-by-command training route.
- `phases/README.md` indexes the project phases and their statuses.
- `phases/progress.md` tracks completed, current, and upcoming work as a checklist.

## Suggested reading order

1. `learning-goals.md`
2. `rpc-learning-path.md`
3. `app-rpc-contract.md`
4. `phases/README.md`
